# Cash — CS50 Week 1

This program calculates the **minimum number of coins needed to make change**.

The program uses a **greedy algorithm** to determine the optimal number of coins.

Example:
Change owed: 0.41
Coins used: 4

## Concepts Used

- greedy algorithms
- loops
- arithmetic operations
