# Readability — CS50 Week 2

This program calculates the **reading level of a text** using the **Coleman–Liau Index**.

The algorithm analyzes:

- number of letters
- number of words
- number of sentences

Then it determines the grade level required to understand the text.

Example output:
Grade 8
