# Caesar — CS50 Week 2

Implementation of the **Caesar cipher**, a simple substitution encryption method.

The program receives a numeric key from the command line and shifts each letter of the plaintext.

## Example

```
./caesar 3
plaintext: HELLO
ciphertext: KHOOR
```

## Concepts used

- command line arguments (`argc`, `argv`)
- character manipulation
- ASCII values
- modulo operator
- functions
- `ctype.h` library
