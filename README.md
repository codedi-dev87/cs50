# CS50 — Programming Exercises (Diogo Ferreira)

This repository contains my solutions to the problem sets from **Harvard's CS50 — Introduction to Computer Science**.

The projects were developed in **C** and cover fundamental programming concepts such as:

- algorithms
- loops
- conditionals
- functions
- string manipulation
- basic cryptography
- text analysis

---

# Repository Structure

## Week 1 — C Basics

Introduction to programming using the C language.

Projects:

- **Mario** — prints pyramids using loops
- **Mario More** — advanced version of the Mario pyramid
- **Cash** — calculates minimum number of coins using a greedy algorithm
- **Credit** — validates credit card numbers using the Luhn algorithm
- **Population Lab** — calculates population growth

---

## Week 2 — Algorithms and Text

Introduction to algorithms and text processing.

Projects:

- **Caesar** — substitution cipher encryption
- **Readability** — calculates text readability using the Coleman–Liau index
- **Scrabble** — calculates Scrabble word scores
